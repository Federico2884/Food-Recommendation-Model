# Food Recommendation Model

Repositori ini berisi proses riset, pengolahan data, dan pelatihan model 
untuk Sistem Rekomendasi Makanan menggunakan Jupyter Notebook.

## Isi Repositori
* `Food_Recommendation_System.ipynb`: Notebook proses training model.
* `healthy_foods_database.csv`: Dataset yang digunakan.

## Implementasi
Model yang dibuat dari notebook ini menggunakan ONNX (Open Neural Network eXchange).
